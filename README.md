# Nabeel Khan | Portfolio (React)

## Overview
This repository contains the source code for my personal developer portfolio built using React. It showcases my projects, technical skills, and development experience in a clean and structured format.

The application is fully responsive and designed with a focus on simplicity, performance, and usability.

## Features
- Single-page application built with React  
- Responsive design for desktop and mobile  
- Structured sections for about, skills, and projects  
- Component-based architecture  
- Easy to customize and extend  

## Tech Stack
